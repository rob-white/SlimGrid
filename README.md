# SlimGrid

A wrapper for [SlickGrid](https://github.com/mleibman/SlickGrid) to slim down the amount of time and code required to create a grid

## A Simple Example

#### Code
```javascript
var data = [
	{'id': 1, 'column': 'value'}, 
	{'id': 2, 'column': 'value'}, 
	{'id': 3, 'column': 'value'}
];

var grid = SlimGrid()
	.container($('body'))
	.render(data);
```
